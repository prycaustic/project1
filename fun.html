I don't know what to put on this page so I am just going to talk about a game that I have been playing recently called Resident Evil 4.

Resident Evil 4 is a survival horror shooter game that was developed by Capcom. It is called RE4, but it is actually the sixth game in the series. The game was released in 2005 for the Nintendo GameCube.

I have been playing the Steam release version with the RE4 HD fan made textures and the game still looks great! I'm even able to use my PS5 Controller to play it.

It features the main character Leon S. Kennedy who is on a mission to rescue the president's daughter who has been kidnapped by a cult in a rural European village.
